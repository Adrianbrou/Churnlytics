# Churnlytics - Frontend

React 18 + Vite 7 SPA for the Churnlytics dashboard. See the [root README](../README.md) for the project overview, stack, and full quick-start.

## Dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production bundle
npm run preview  # preview the production bundle
```

Expects the Flask API on `http://localhost:5000`. Start it from `../backend/`.
